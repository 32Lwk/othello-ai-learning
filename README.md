# オセロAI学習システム

Q学習を使用したオセロAIの実装です。人間vsAI対戦やAI同士の訓練を通じて、AIの学習進捗を可視化できます。

## 機能

- **人間vsAI対戦モード**: 人間プレイヤーとAIが対戦し、AIが学習
- **AI同士訓練モード**: AI同士で対戦させて事前訓練
- **学習進捗可視化**: 勝率、平均報酬、学習回数の推移をグラフ表示
- **学習データ管理**: 学習データの保存・読み込み・削除機能
- **対戦記録**: 詳細な対戦履歴の表示

## セットアップ

### 必要な環境
- Python 3.7以上
- pygame

### インストール手順

1. リポジトリをクローン
```bash
git clone https://github.com/32Lwk/othello-ai-learning.git
cd othello-ai-learning
```

2. 必要なライブラリをインストール
```bash
pip install pygame
```

## 使用方法

### メインゲームの実行
```bash
cd 学習オセロ
python オセロ2.py
```

### ゲームモード

1. **人間vsAIで学習**: 人間プレイヤーとAIが対戦し、AIが学習
2. **AI同士で訓練→人間vsAI**: AI同士で事前訓練後、人間と対戦
3. **学習データ参照**: 保存された学習データの確認
4. **対戦記録参照**: 詳細な対戦履歴の表示

### 操作方法

- **マウスクリック**: 盤面に石を配置
- **リセットボタン**: 新しいゲームを開始
- **戻るボタン**: モード選択画面に戻る

## 学習データの管理

学習データは以下の形式で保存できます：

- **Qテーブル**: `qtable_<名前>.pkl`
- **学習履歴**: `learning_history_<名前>.json`

### データ管理機能

- **データ保存**: 現在の学習状態を名前付きで保存
- **データ読み込み**: 保存された学習データを読み込み
- **新規作成**: 新しい学習データを作成
- **データ削除**: 不要な学習データを削除

## AI学習レベル

AIの学習レベルは以下の要素で計算されます：

- **勝率**: 対戦での勝率
- **平均報酬**: 1手あたりの平均報酬
- **学習回数**: 総学習回数
- **Qテーブルサイズ**: 学習した状態数

レベルは1〜10段階で表示され、レベルに応じた称号が付与されます。

## 注意事項

- 日本語ファイル名を使用しているため、Windows環境での実行を推奨
- 学習データファイル（`.pkl`、`.json`）は`.gitignore`で除外されているため、必要に応じて手動でバックアップ
- 大きなファイル（GIF、画像、フォントファイルなど）は除外されています
